== Instruction ==

== Update == 
At each plugin update, go to the WPaptouch setting page. There you have nothing to do, just check if all setting is okay, you just trigger a action to help ioswp update correctly visiting the setting page.

== It doesn't work? == 
ioswp plugin create a folder in your Wordpress themes folder ('/wp-content/themes/wp_ioswp'). if after the installation this folder is not created automatically, create a folder on your server with free FTP software like filezilla. Name it wp_ioswp, copy these 3 files from your wp_ioswp plugin folder (index.html, style.css, screenshot-1.jpg) to the themes folder you just named wp_ioswp. Rename the file extention index.html to index.php 
Return to the ioswp setting, from 'Alternative themes to be displayerd' select the wp_ioswp theme.

== How can i change the themes color and design? == 
ioswp will come with a pro version with many more options like: Color options, mobile ads, users Comments, favorites articles, (*Phonegap integration)

* May add this option if there's a demand for it.

== Installation ==

1. Upload the folder wioswp in the zip file to your plugins directory. ('/wp-content/plugins/')
2. Activate the plugin from your Plugins page.
3. Add image to your posts. Go to the setting page to set options.
